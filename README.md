# Jacksen & Jessica's Wedding Invitation

## Features
- Beautiful and fully responsive digital wedding invitation.
- Personalized guest greeting via URL query parameter (e.g., `?guest=Budi`).
- Smooth **welcome section fade-out** animation revealing main content.
- RSVP form integrated with Firebase Realtime Database.
- Real-time message board for guests to leave greetings.
- Easy to customize text, images, and venues.
- Works on desktop and mobile devices.
